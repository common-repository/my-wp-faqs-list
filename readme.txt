== Installation ==

1. Install as a regular WordPress plugin
2. You have to use shortcode for generate Awesome FAQs list  in post or page. Example shortcode is given below.
<strong>Use WP FAQs list shortcode</strong>
<pre>[faq]</pre>

<strong>Here can change</strong>
<pre>[faq post_type="" category="" count=""]</pre>
